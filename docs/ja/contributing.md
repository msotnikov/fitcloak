# コントリビュート

## 開発

```bash
./gradlew build    # ビルド
./gradlew test     # テスト実行
./gradlew run      # サーバー起動
```

Java 17+ が必要です。

## ガイドライン

開発ガイドラインとコントリビューションのワークフローについては [CONTRIBUTING.md](https://github.com/msotnikov/fitcloak/blob/main/CONTRIBUTING.md) を参照してください。
