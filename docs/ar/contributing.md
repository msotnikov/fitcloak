# المساهمة

## التطوير

```bash
./gradlew build    # البناء
./gradlew test     # تشغيل الاختبارات
./gradlew run      # تشغيل الخادم
```

يتطلب Java 17+.

## الإرشادات

انظر [CONTRIBUTING.md](https://github.com/msotnikov/fitcloak/blob/main/CONTRIBUTING.md) لإرشادات التطوير وسير عمل المساهمة.
