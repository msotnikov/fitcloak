# 贡献

## 开发

```bash
./gradlew build    # 构建
./gradlew test     # 运行测试
./gradlew run      # 启动服务器
```

需要 Java 17+。

## 指南

开发指南和贡献工作流请参见 [CONTRIBUTING.md](https://github.com/msotnikov/fitcloak/blob/main/CONTRIBUTING.md)。
