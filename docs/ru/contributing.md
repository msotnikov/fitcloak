# Участие в проекте

## Разработка

```bash
./gradlew build    # Сборка
./gradlew test     # Запуск тестов
./gradlew run      # Запуск сервера
```

Требуется Java 17+.

## Руководство

См. [CONTRIBUTING.md](https://github.com/msotnikov/fitcloak/blob/main/CONTRIBUTING.md) для руководства по разработке и процессу внесения вклада.
