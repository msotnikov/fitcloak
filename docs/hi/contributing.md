# योगदान

## विकास

```bash
./gradlew build    # बिल्ड
./gradlew test     # टेस्ट चलाएँ
./gradlew run      # सर्वर शुरू करें
```

Java 17+ आवश्यक है।

## दिशानिर्देश

विकास दिशानिर्देशों और योगदान कार्यप्रवाह के लिए [CONTRIBUTING.md](https://github.com/msotnikov/fitcloak/blob/main/CONTRIBUTING.md) देखें।
