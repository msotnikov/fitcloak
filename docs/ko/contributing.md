# 기여하기

## 개발

```bash
./gradlew build    # 빌드
./gradlew test     # 테스트 실행
./gradlew run      # 서버 시작
```

Java 17+ 필요.

## 가이드라인

개발 가이드라인과 기여 워크플로우에 대해서는 [CONTRIBUTING.md](https://github.com/msotnikov/fitcloak/blob/main/CONTRIBUTING.md)를 참조하세요.
