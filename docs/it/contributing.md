# Contribuire

## Sviluppo

```bash
./gradlew build    # Build
./gradlew test     # Run tests
./gradlew run      # Start server
```

Richiede Java 17+.

## Linee guida

Consulta [CONTRIBUTING.md](https://github.com/msotnikov/fitcloak/blob/main/CONTRIBUTING.md) per le linee guida di sviluppo e il flusso di lavoro per contribuire.
